## Getting Started

1. Clone the repository
```bash
git clone https://github.com/your-username/chizo-portfolio.git
```

2. Install dependencies
```bash
cd chizo-portfolio
npm install
```

3. Start the development server
```bash
npm run dev
```

4. Build for production
```bash
npm run build
```
